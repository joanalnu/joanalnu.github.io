Christmas Greeting Generator

What You Will Learn

  • How to store messages in Python lists
  • How to pick elements randomly
  • How to print variables

Step 1: Create a File

Call it greeting_generator.py.

Step 2: Create a List with Holiday Greetings

greetings = [ "Merry Christmas!", "Happy Holidays!", "Warm winter wishes!", "Have a magical Christmas season!", "May your day be merry and bright!" ]

Remember! Python lists are groups of elements separated by commas and written between []. They can contain repeated elements and mixed variable types.

Step 3: Use random.choice() to Pick a Greeting

import random message = random.choice(greetings) print(message)

Step 4: Run It!

Each time you run the program, a new greeting appears!

üéØ Challenges

  • Add more greetings
  • Ask the user for their name and include it in the message

üí° Advanced Tip

Test whether your computer can generate true randomness by comparing coin flips or dice rolls with expected frequencies!

import random # Coin flip example frequency = 0 flips = 100 for i in range(flips): coin = random.choice([0, 1]) frequency += coin print(f'Heads: {frequency}, Tails: {flips-frequency}')

ASCII Christmas Tree

What You Will Learn

  • How to use loops
  • How to build shapes with text

Step 1: Create the File

Call it ascii_tree.py.

Step 2: Ask for Tree Height

height = int(input("How tall should your tree be? "))

This takes input from the user and converts it to an integer. The default type from input() is a string!

Step 3: Build the Tree with a Loop

for i in range(1, height + 1): spaces = " " * (height - i) stars = "*" * (2 * i - 1) print(spaces + stars)

Can you work out how many characters are printed in this loop?

Step 4: Add a Trunk

print(" " * (height - 1) + "|")

Step 5: Run It!

Try different heights: 3, 5, 10...

üéØ Challenges

  • Use # instead of *
  • Add ornaments as random characters
  • Add color using ANSI codes

üí° Advanced Tip: Create a Snowflake!

height = int(input("How big should your snowflake be? ")) for i in range(-height, height + 1): line = "" for j in range(-height, height + 1): if abs(i) == abs(j) or i == 0 or j == 0: line += "*" else: line += " " print(line)

Hot Chocolate Recipe

What You Will Learn

  • Dictionaries in Python

What You Will NOT Learn

This is not a real recipe, don't try to prepare it!!

Step 1: Create the File

Call it hot_chocolate_recipe.py.

Step 2: Create the Recipe Information

ingredients = { 'chocolate powder': 30, 'sugar': 5, 'milk': 200, }

A dictionary enables you to create paired values: keys and values.

Step 3: Ask for the Number of People

people = int(input("How many people are going to drink? "))

Step 4: Calculate Amounts

print('You are going to need...') for ingredient in ingredients.keys(): amount = people * ingredients[ingredient] print(f'{amount} weight units of {ingredient}')

Use dictionaries like value = dict_name[key] to access values.

Step 5: Run It!

Try different group sizes!

üéØ Challenges

  • Add different recipes with a recipe selector

Snowball Gift

What You Will Learn

  • Basic geometry
  • Formulas with Python

Step 1: Create the File

Call it snowball_gift.py.

Step 2: Ask for the Snowball Radius

radius = int(input("What is the radius of your snowball? "))

Step 3: Compute the Volume

import math snowball_volume = (4 * math.pi * (radius**3)) / 3 print(f'The volume is {snowball_volume} cubic length units.')

The result is in cubic length units since you can enter the radius in any unit!

Step 4: How Much Paper to Wrap It?

The box is a cube with sides of length 2r to fit the snowball perfectly.

a = 2 * radius # length of 1 side side_area = a * a # area of 1 side cube_area = 6 * side_area # a cube has 6 sides print(f"You'll need {cube_area} squared length units!")

Question: Why isn't cube_area equal to a¬≥? What would that expression describe?

Step 5: Run It!

Try different radii!

üéØ Challenges

  • How much would the snowball weigh? (water density is 1)
  • How many snowballs with radius r/2 fit in a box for radius r?
