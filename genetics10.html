About Genetics10

Genetics10 is a Python-based tool designed for students to explore biology and genetics concepts such as DNA transcription, mutations, protein structures, and gene editing. Built using Jupyter Notebook, the software is open-source and web-accessible without requiring any installation, making it ideal for classroom use. It is also designed to be flexible and adapt to different students, with or without prior knowledge of coding.

Key Features

  • DNA to RNA transcription and amino acid sequencing
  • Simulate genetic mutations
  • Compare DNA sequences
  • Visualize protein structures using AlphaFold API integration
  • Gene editing with CRISPR-Cas9
  • Open-source and easily scalable and customizable

API Integration

We are excited to introduce the Genetics10 API, a powerful tool that brings all the features of our notebook directly into your own code. Whether you're building web apps, conducting advanced research, or teaching at high school, the API provides seamless integration of the mentioned functionalities right where you need them. This API enables developers, researchers, and educators to integrate the genetics10 module seamlessly with their own and tailored needs.

Info for Educators

Teachers can use Genetics10 to enhance biology lessons by creating assignments and problem-solving activities that require students to explore real-world biological processes like genetic mutations and transcription. This tool is fully aligned with general education guidelines.

For instance, you can implement the flipped classroom method, assigning tutorials for homework and using class time for interactive activities. Additionally, group projects can be designed to encourage collaboration and achieve great milestones.

Genetics10 is now available in English, Spanish, German, and Catalan. Further translations are in the works, and other languages can be made on demand.
