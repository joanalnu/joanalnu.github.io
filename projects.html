    The CAPIBARA Project is a student-led initiative aiming to launch a satellite dedicated to observing and researching the high-energy cosmos. Our mission focuses on measuring primary cosmic rays (charged particles) as well as X-ray and ùõæ-ray emissions, providing valuable insights into the Universe's most energetic phenomena.

    By studying high-energy radiation, we aim to deepen our understanding of astrophysical processes, quantum physics, and the interactions between cosmic phenomena and Earth's space environment.     Genetics10 & GEN-API

    ×

    Genetics10 is a free, web-based, educational software for teaching biology and genetics. It's a Python Notebook, where students can easily play and make their code using the pre-established functions. Furthermore, advanced students can write more complex code and create other functions. The possibilities are endless! Additionally, it is web-based and does not require any installation at all, making it very easy to use and perfect for school-managed devices.

    GEN-API is an open-source API for genetic data analysis. It features the same functions as the original gen10 class in all the available languages, but it can be installed in your computer using pip and used system-wide. This enables a smoother integration for external projects, more complex data analysis and integration with other tools. It is also very easy to use and can be installed in a few seconds.
