About Genetics11

Gen11 is a Python package designed for students to explore genetics and biology concepts. This package provides useful functions ready-to-use for students to focus on the learning.

It requires no software or app installation, making it accessible and classroom-friendly, perfect for school managed devices.

In short, Gen11 provides the genomics functions you need right at your fingertips.

gen11_example.py 
import gen11

# DNA transcription
dna = "ATCGATCG"
rna = gen11.transcribe(dna)

# Protein synthesis
protein = gen11.translate(rna)

# Visualize results
gen11.visualize_protein(protein)
