    Acerca de Genetics10

    Gen10 es un paquete de Python diseñado para que los estudiantes exploren conceptos de genética y biología. Este paquete proporciona funciones útiles listas para usar para que los estudiantes se enfoquen en el aprendizaje. Gen10 es una herramienta de código abierto, escalable y personalizable para adaptarse a todos los estudiantes y niveles de experiencia.

    No requiere instalación de software o aplicaciones, lo que la hace accesible y amigable para el aula, perfecta para dispositivos gestionados por la escuela. Puede importarse en cualquier código Python (scripts y notebooks) y usarse sin problemas con código ya existente o para crear nuevos programas.

    En resumen, Gen10 ofrece las funciones genómicas que necesitas al alcance de tu mano.

    Características Clave

    • Transcripción de ADN a ARN y secuenciación de aminoácidos
    • Simulación de mutaciones genéticas aleatorias o mutaciones en sitios específicos
    • Comparación de secuencias de ADN
    • Encontrar complementarios y contenido de GC
    • Iteración fácil a través de conjuntos de datos
    • Visualización de estructuras proteicas usando integración con la API de AlphaFold
    • Edición genética simulando CRISPR-Cas9
    • Simulación de una PCR
    • Lectura y escritura de archivos FASTA y GenBank
    • Estimación de temperaturas de fusión (para <14 bases)

    Para Estudiantes

    ¿Nuevo en Python? ¿O nuevo en programación en general? Comienza con la completa Guía para Principiantes en este Notebook escrita por estudiantes como tú. Aprende a tu propio ritmo y desbloquea el potencial de los métodos de Gen10 a través de la exploración práctica de bioinformática. ¡Recuerda que esta herramienta está diseñada para ti, así que los errores no existen!

    Para Educadores

    Gen10 es la herramienta perfecta para el aula para enseñar genética de forma interactiva y atractiva. Soporta actividades grupales, aulas invertidas y está alineada con el currículo (al menos comenzó así antes de añadir más métodos). Está disponible en varios idiomas (inglés, español, alemán, catalán) con idiomas adicionales posibles bajo petición. Consulta la Información para Educadores en la documentación de GitHub.
