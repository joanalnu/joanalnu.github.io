TdM - Tables de Multiplication

Bonjour ! Bienvenue sur le site officiel de TdM. TdM est un programme qui vous aide à pratiquer les mathématiques et à rendre l'apprentissage amusant. Il s'agit d'un code basé sur le navigateur que n'importe qui peut utiliser, vous n'avez qu'à cliquer sur le bouton « JOUER » ci-dessous.

De plus, TdM est un projet open-source qui s'améliore constamment. Nous vous encourageons à contribuer en signalant des problèmes ou en envoyant des demandes de tirage sur le référentiel GitHub du projet. J'espère que vous l'apprécierez !
