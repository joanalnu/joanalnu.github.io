TdM - ตารางคูณ

สวัสดี! ยินดีต้อนรับสู่เว็บไซต์อย่างเป็นทางการของ TdM TdM เป็นโปรแกรมที่ช่วยให้คุณฝึกคณิตศาสตร์และทำให้การเรียนรู้สนุก มันเป็นโค้ดที่ใช้งานได้บนเว็บที่ทุกคนสามารถใช้งานได้บนเบราว์เซอร์ คุณเพียงแค่ต้องคลิกปุ่ม "เล่น" ด้านล่าง

นอกจากนี้ TdM เป็นโครงการโอเพ่นซอร์สที่กำลังพัฒนาและดีขึ้นอย่างต่อเนื่อง เราเชิญชวนให้คุณมีส่วนร่วมในการพัฒนาโครงการโดยการรายงานปัญหาหรือส่งคำขอ pull request ไปยัง GitHub repository ของโครงการ เราหวังว่าคุณจะชอบมัน!
