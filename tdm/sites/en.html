TdM - Tablas de Multiplicar

Hello! Welcome to TdM's official website. TdM is a program that helps you practice math and makes learning fun. It's a web-based code that anyone can use on the browser, you only have to click the "PLAY" button below.

Additionally, TdM is an open-source project and is always getting better. We encourage you to contribute either by reporting issues or sending pull requests on the project's GitHub repository. I hope you enjoy it!
