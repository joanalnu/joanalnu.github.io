TdM - Tabelas de Multiplicação

Olá! Bem-vindo ao site oficial do TdM. O TdM é um programa que ajuda a praticar matemática e torna o aprendizado divertido. É um código baseado na web que qualquer pessoa pode usar no navegador, basta clicar no botão "JOGAR" abaixo.

Além disso, o TdM é um projeto de código aberto e está sempre melhorando. Encorajamos você a contribuir relatando problemas ou enviando solicitações de pull no repositório do GitHub do projeto. Espero que você goste!
